# Beispiele

Dieses Verzeichnis enthält Beispiel-Vorsorgeausweise und typische Beratungsszenarien.

## Struktur

```
examples/
├── vorsorgeausweise/    # Anonymisierte Vorsorgeausweise (BVG)
├── scenarios/           # Beratungsszenarien
└── calculations/        # Beispielberechnungen
```

## Datenschutz

⚠️ **Wichtig**: Alle Beispiele müssen vollständig anonymisiert sein!

- Keine echten Namen
- Keine echten AHV-Nummern
- Keine echten Adressen
- Typische, aber fiktive Werte

## Verwendung

Diese Beispiele dienen:
- Als Testdaten für die Entwicklung
- Als Benchmark für Berechnungen
- Als Demonstrations-Material
- Zur Validierung der Skill-Genauigkeit

## Nächste Schritte

Beispieldaten werden während der Entwicklung der Skills erstellt (ab Phase 3).
